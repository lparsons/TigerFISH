Spot Counts

spot_counts.tsv

Tab-delimited file with the following columns:

  • 'Experiment' - Name of experiment
  • 'Region' - Region number
  • 'Cell' - Cell Number (0=background)
  • 'Cy3' - Spot count for Cy3
  • 'Cy3.5' - Spot count for Cy3.5
  • 'Cy5' - Spot count for Cy5
  • 'Cell Phase' - Cell phase as determined by DAPI intensity
    • -1 -> No phase, background, not cell
    • 0 -> other?
    • 1 -> G1
    • 2 -> S
    • 3 -> G2
    • 4 -> other?
    • 5 -> other?
